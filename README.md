# 🧮 Calculator - Gesture Based

A unique calculator application controlled by hand gestures.

---

## 🌟 Overview

Perform mathematical calculations using intuitive hand gestures captured via webcam.

## ✨ Features

- **Gesture Input** - Touch-free control
- **Visual Feedback** - Real-time tracking overlay
- **Standard Math** - Basic arithmetic operations

## 📁 Project Structure

```
calculator/
├── main.py              # Entry point
├── calculator_gui.py    # UI Implementation
├── gesture_detector.py  # Hand tracking logic
└── requirements.txt     # Dependencies
```

## 🚀 Quick Start

```bash
pip install -r requirements.txt
python main.py
```
