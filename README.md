# Gesture Calculator

Calculator controlled entirely by hand gestures in the air, using computer vision to recognize numeric and operator inputs.

## Features

- Hand gesture recognition for digit input (0-9)
- Gesture mapping for arithmetic operators
- Real-time calculation display
- Gesture history and expression building

## Tech Stack

Python, OpenCV, MediaPipe, NumPy

## Getting Started

### Prerequisites

- Python 3.8+
- pip

### Installation

```bash
git clone https://github.com/karthik-idikuda/Gesture-Calculator.git
cd Gesture-Calculator
pip install -r requirements.txt
```

### Usage

```bash
python gesture_calc.py
```
